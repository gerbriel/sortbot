# Description Formatting Example

## Input:
- Voice: "megadeath world tour concert 1990s grail t-shirt"
- Brand: "Megadeath"
- Era: "Vintage / Y2K"
- Size: "Large"
- Measurements:
  - Pit to Pit: 20
  - Length: 27
- Color: "Black/White"
- Category: "T-Shirt"

```

✠ SIZE- Large
✠ Pit to pit- 20
✠ length- 27

BUNDLE AND SAVE!!!!!!

```

## Format Structure:

1. **Main Description** (from voice)
   - Uses your exact words, capitalized

2. **Measurements Section** (with ✠ symbols)
   - SIZE
   - Pit to pit
   - Length
   - Any other measurements

3. **Call to Action**
   - "BUNDLE AND SAVE!!!!!!"

4. **Tags** (with # hashtags)
   - Auto-generated from: brand, era, category, colors, size

5. **Standard Disclaimers**
   - Vintage character note
   - High-quality piece
   - Next-day shipping
   - All sales final

## Usage:

1. Upload images
2. Group them
3. Add voice description (e.g., "megadeath world tour 1990s grail t-shirt")
4. Fill in size, measurements, brand, era
5. Click "✨ Generate AI Description"
6. Description auto-formatted in vintage streetwear style!

The system will:
- Keep your voice description as the main text
- Format measurements with ✠ symbols
- Add BUNDLE AND SAVE
- Generate hashtags
- Add standard disclaimers
